# 📚 How To Use



```bash
# Clonar o repositório
git clone https://github.com/MaximianoEduardo/Spotify-Clone.git
```

```bash
# Instalando Dependências
yarn install
# or
npm install

# Run 
yarn start
# or
npm start
```
